Lab 3

For this lab, you will demonstrate your knowledge of responsive web design for navigation bars as well as JavaScript. This lab will be submitted via github. Watch the Week 2 videos for more information on how to work with github.

Follow these instructions to complete the lab:

    1. Create a page that has a header, main, and footer section. The header and footer should take up a fixed amount of height while the main element will take up the rest. Make sure the page spans at least 100% of the height of the viewport.

    2. Create responsive navigation that shows as a horizontal navigation bar between the header and main sections on desktop, while on mobile it shows as a navigation tab bar affixed to the bottom of the viewport.

    3. Create four navigation items with font awesome icons and text each representing a different color of your choosing. When the user clicks (or touches) one of the navigation items, change the background color of the main element to that navigation item's represented color.

Rubric

(5 points) header, main, and footer sections in correct order that each have at least a different color background

(5 points) header and footer are fixed heights while main section takes up remaining height

(10 points) desktop (769px+ width) navigation bar shows horizontally between header and main

(10 points) mobile navigation bar (<=768px width) shows affixed to bottom of viewport regardless of scrolling

(5 points) clicks on navigation items change the main background color

(5 points) main background color change accomplished via addEventListener and adding and removing css classes to the classList

(5 points) navigation items use font-awesome

(5 points) four navigation items

(10 points) your animated name from lab 1 is in the header and resizes appropriately for mobile devices

(-5 points per style attribute) Don't use inline styles
