Weather Plus php · html · css · api's

Displays current weather info according to location from DarkSky.net's data; makes use of Mapquest API for geocoding/geolocation; unique features including a moon phase display according to the exact lunation number, dimming of the background according to local daylight, and a flashing UV warning; coordinates displayed link to Google Maps search; challenges in putting this together arose around dealing timezone and other date-time issues; much was learned about the limitations and advantages of doing different things client-side or server-side; in the future I would like to add dynamic weather effects as well, such as css animated rainfall and clouds; in the process of implementing a mobile web version at this time
